# Color Picker Utility

> A simple, lightweight color picker utility built as part of the [100LinesOfCode](https://github.com/josharsh/100LinesOfCode) project.

This tool provides an interactive way for developers and designers to select a color and instantly get its value converted across common web formats.

## Features

* Interactive color selection (`<input type="color">`)
* Real-time conversion to **HEX**, **RGB**, and **HSL**
* "Copy to clipboard" functionality for each format

## Tech Stack

* **HTML:** For the structure.
* **CSS:** For styling.
* **Vanilla JavaScript:** For all logic (kept under 100 lines).
